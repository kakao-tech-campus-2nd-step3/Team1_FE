# 코드 컨벤션

- angular code conventions을 기반으로 커밋 메세지 작성
    - feat : 새로운 기능 추가
    - fix : 버그 수정
    - docs : 문서 변경
    - style : 코드 스타일 변경 (포매팅 수정, 세미콜론 추가 등)
    - refactor : 코드 리팩토링
    - test : 테스트 코드 추가, 수정
    - chore : 빌드 프로세스, 도구 설정 변경 등 기타 작업
